## 🌐 Deployment

This site is a static website and can be deployed for **free** on any of the following platforms:

### GitHub Pages

1. Push code to your GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch → / (root)**
4. Your site will be live at `https://ashokml1729.github.io/portfolio`

### Vercel

1. Import your GitHub repo at [vercel.com](https://vercel.com)
2. No configuration needed — Vercel auto-detects static sites
3. Your site will be live at `https://portfolio-ashok.vercel.app`

### Netlify

1. Drag and drop your project folder at [netlify.com](https://netlify.com)
2. Instant deploy — no setup required
